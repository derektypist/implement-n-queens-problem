# Implement N-Queens Problem

The aim is to place N queens on an N x N chessboard so that no two queens attack each other (no two share a row, column, or diagonal).

## Technologies

Uses Python3.

## Deployment

Copy `main.py` into the text editor of your choice.  If using terminal, use the command `python3 main.py` and press `Enter` to run the code.
